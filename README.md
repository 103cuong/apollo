# thinid

> 🚀📦 Easy way to create 🆔 unique id.

ThinId creates lovely unique ids. Easy to use, support both JavaScript and TypeScript.

## Installation

```sh
yarn add thinid
# or
npm install -s thinid
```

## Usage

```javascript
import thinid from 'thinid';

console.log(thinid());
```

```sh
Output should be '171a7-0688a_69261'
```
