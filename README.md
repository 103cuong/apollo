# thinid

🚀📦 Create 🆔 unique ID like eating 🥞 pancakes.

## Installation

```sh
yarn add thinid
# or
npm install -s thinid
```

## Usage

```javascript
import thinid from 'thinid';

console.log(thinid());
```

```sh
Output should be '3ywbk3e0-7825-e798-90ee-948s2lmw82bd'
```
